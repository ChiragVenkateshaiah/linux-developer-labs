# linux-developer-labs
This repository contains **two structured projects** designed to build **production-ready Linux skills** through **theory + hands-on labs** executed on cloud VMs

---

## Project 1: Core Linux Foundations (Labs + Theory)
To build **strong Linux fundamentals** by combining:
- **Conceptual understanding (theory)**
- **Hands-on execution (labs)**
- **Real VM environments (GCP)**

---

## Learning Philosophy
For every topic:
- Why it exists
- How Linux implements it
- How engineers use it in real systems
- Hands-on validation on a VM

## Topic Covered (High Level)
| Day   | Focus Area                             |
| ----- | -------------------------------------- |
| Day 1 | Linux boot process, PID 1, `ps`, `top` |
| Day 2 | Signals, `kill`, process lifecycle     |
| Day 3 | Filesystem hierarchy, permissions      |
| Day 4 | `apt`, packages, system services       |
| Day 5 | Networking (`ip`, `ss`, ports)         |
| Day 6 | CPU, memory, disk monitoring           |

## Environment
- **Cloud: GCP**
- **OS: Ubuntu Server(Minimal)**
- **Access: SSH (no GUI)**
- **Focus: Server-side Linux**

## Outcome of Project 1
- Understanding Linux internals confidently
- Debug system issues
